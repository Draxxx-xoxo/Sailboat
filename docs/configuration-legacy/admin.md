# Admin

## Configuration

| Option   | Description                               | Type   | Default |
| -------- | ----------------------------------------- | ------ | ------- |
| lang     | what language you want the bot to display | String | en      |
| prefix   | Bot prefix                                | String | null    |
| nickname | A nickname for the bot                    | String | null    |

Example

```
guild_settings:
  lang: en
  prefix: '!'
  nickname: wall-ellll
```

## Commands
